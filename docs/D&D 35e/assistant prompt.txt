# TTRPG Structural Analysis Matrix v2

## Mapping for 17 Reference Games

| Category | Contains | Boundary Rule |
|----------|----------|---------------|
| meta | Sheet/system metadata | Would change if porting to different system/campaign |
| identity | Biographical, descriptive | Static description with no direct mechanical rating |
| attributes | Core innate stats | Fundamental ratings, rarely change through play |
| skills | Learned/developed ratings | Improve through use/advancement, rolled for actions |
| resources | Pools, tracks, conditions | Fluctuates during play; spent/recovered |
| features | Discrete capabilities | You have it or you don't (though may have sub-properties) |
| inventory | Physical possessions | Tangible things the character owns |
| connections | Relationships to entities | Links to NPCs, factions, PCs, organizations |
| narrative | Story with mechanical hooks | Text that gets USED mechanically during play |
| progression | Advancement state | Tracks growth; actual gains go to their respective categories |

### Key Disambiguation Rules

Attribute vs Skill:
- If it represents innate capability that defines the character fundamentally → Attribute
- If it represents learned/practiced capability that improves through use → Skill
- If a game doesn't distinguish, default to Skills

Skill vs Feature:
- If it has a rating you roll/reference → Skill
- If it's binary (have/don't have) with defined effects → Feature
- If it's binary BUT has a rating attached, it's a Feature with a rated sub-property

Resource vs Attribute:
- If it depletes during a session and recovers → Resource
- If it rarely changes except through advancement → Attribute

Narrative vs Identity:
- If it gets mechanically invoked during play → Narrative
- If it's pure description/flavor → Identity

Feature vs Narrative:
- If it has defined mechanical effects → Feature
- If it provides story hooks/fictional positioning → Narrative
- (Some elements are both; they go in Narrative with mechanical properties)

## Structural Pattern Analysis

### Attribute Representation Patterns

```
NUMERIC VALUE (add to roll)
├── Moderate range (-3 to +5): Ars Magica, Apocalypse World
├── Large range (3-18+): Pathfinder, CoC
└── Abstract (1-20+): GURPS

DICE TYPE (roll the die)
├── Single step die: Kids on Bikes
├── Paired step dice: Ryuutama, T2000
└── Die = pool size: VtM (Attribute dots = dice)

DOTS/PIPS (count for pool)
└── VtM, Scum and Villainy

POOL IS VALUE (dual-purpose)
└── Cypher (Might/Speed/Intellect are pools AND attributes)
```

---

### Skill Representation Patterns

```
PERCENTILE
└── CoC (0-99, with derived half/fifth)

NUMERIC RANKS
├── Uncapped: Burning Wheel, Ars Magica
├── Level-capped: Pathfinder
└── Abstract cost: GURPS

DIE TYPE
└── T2000

DOTS/PIPS
└── VtM, S&V

LADDER (abstract levels)
└── Fate

BINARY TIER (trained/untrained/specialized)
└── Cypher

POOL POINTS (current/max spendable)
└── GUMSHOE/Sword of Serpentine

FREEFORM (any name valid)
└── Troika!
```

---

### Resource Track Patterns

```
SIMPLE POOL (current/max)
├── HP: Pathfinder, Ryuutama, CoC
├── MP: Ryuutama
└── Sanity: CoC

DUAL-TRACK POOL (two damage types)
└── VtM (Superficial / Aggravated)

BOXES (fill left to right)
├── Stress: S&V
├── Harm levels: S&V
├── Stress boxes: Fate
└── Recovery uses: Cypher

LABELED LEVELS (each has meaning)
├── Fatigue: Ars Magica
├── Wounds: Burning Wheel
└── Consequences: Fate (with aspect text)

THRESHOLD (current vs. max, different effect)
├── Wounds/Strain: Genesys
└── Stress threshold: T2000

DEGRADING DIE
└── Ammo: T2000

POOL-AS-HEALTH (same resource is pool and HP)
└── Cypher

INVERSE POOL (never goes up, rarely recovers)
├── Luck: CoC
└── Luck: Troika!
```

---

### Feature Patterns

```
BINARY (have or don't)
├── Feats: Pathfinder
├── Moves: Apocalypse World
├── Talents: T2000
└── Stunts: Fate

BINARY WITH RATING (have it, and it has a level)
├── Discipline powers: VtM (level in discipline)
├── Advantages: GURPS (point cost = intensity)
└── Tier abilities: Cypher

BINARY WITH PREREQUISITES
├── Feats: Pathfinder (chains)
├── Talents: Genesys (trees)
└── GURPS (prerequisite advantages)

BALANCED SET (must take equal positive/negative)
├── Virtues/Flaws: Ars Magica
└── Advantages/Disadvantages: GURPS (point balance)

PLAYBOOK-GRANTED (package deal)
├── Moves: Apocalypse World
├── Abilities: Scum and Villainy
└── Type/Focus abilities: Cypher

CUSTOM-DEFINED (player writes the effect)
└── Stunts: Fate
```

---

### Narrative Element Patterns

```
INVOKABLE PHRASE (use for bonus)
└── Aspects: Fate

COMPELLABLE PHRASE (accept for reward)
└── Aspects: Fate (compels give FP)
└── Disadvantages: GURPS (worth points for causing problems)

ADVANCEMENT TRIGGER (pursuing grants XP/meta-currency)
├── Beliefs: Burning Wheel (Artha)
├── Drives: Sword of Serpentine (XP)
└── XP triggers: Scum and Villainy

SANITY ANCHOR (protect against mental harm)
└── Backstory elements: CoC

RATED PERSONALITY (can be rolled)
└── Personality traits: Ars Magica

GOAL TRACKING (complete for reward)
├── Beliefs: Burning Wheel
├── Ambition/Desire: VtM
└── Big Dream: T2000
```

---

### Progression Patterns

```
XP → LEVEL (threshold unlocks package)
├── Pathfinder
└── Ryuutama

XP → SPEND (point-buy improvements)
├── GURPS
├── Genesys
├── Cypher
├── VtM
└── Sword of Serpentine

USE-BASED (improve what you use)
├── Burning Wheel (test types)
├── CoC (skill checks)
├── T2000 (marks)
└── Troika! (advanced skills)

TRIGGER-BASED (specific actions grant XP)
├── S&V (playbook triggers)
└── Apocalypse World (highlighted stats)

MILESTONE (story beats grant improvements)
├── Fate
└── Ars Magica (seasonal, but tied to in-game time)

TRAIT VOTE (group decides)
└── Burning Wheel (end of arc)
```
